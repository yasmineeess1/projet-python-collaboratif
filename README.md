# projet-python-collaboratif
To do List  
1️⃣ Présentation du projet
Ce projet consiste à développer une application de gestion de tâches (To-Do List) en Python. L’objectif est de permettre à un utilisateur d’ajouter, supprimer, modifier et marquer des tâches comme terminées. L’application offrira une interface conviviale et intuitive pour une meilleure gestion du temps et des priorités.

2️⃣ Technologies utilisées
Langage : Python
Bibliothèques : Tkinter (interface graphique), SQLite (stockage des tâches), datetime (gestion des dates)
3️⃣ Fonctionnalités principales
 Ajouter une nouvelle tâche avec une date limite
 Afficher la liste des tâches en cours et terminées
 Modifier ou supprimer une tâche
 Marquer une tâche comme terminée
 Sauvegarde des tâches dans une base de données

4️⃣ Objectifs du projet
Automatiser la gestion des tâches
Offrir une interface utilisateur simple et efficace
Assurer la persistance des données via SQLite
5️⃣ Évolution possible
 Ajout d’une fonctionnalité de rappels pour les échéances
 Synchronisation avec un service en ligne (Google Tasks, API REST)
